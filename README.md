## 数据库格式
### redis
**配置要抓取的币种**
- key:<市场名称>-CONFIG
- value:[[<交易币种>-<计价币种>], [<交易币种>-<计价币种>]...]

**深度行情信息**
- key:<交易币种>-<计价币种>-<市场名称>-DEPTH
- value:{<br>
"up_time" 更新时间</br>
"forbuy" 买盘数据[[<价格>,<数量>],...]</br>
"forsell" 卖盘数据[[<价格>,<数量>],...]</br>
}
